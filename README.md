# 커스텀 플러그인

## 설치방법

<details>
<summary>설치방법</summary>
  <p>

  ```
  cd privates
  git clone {git repository url}
  ```

  <br>

  XpressEngine3이(가) 설치된 루트로 이동한 후 아래 명령어로 플러그인을 설치하세요.

  ```
  php artisan plugin:private_install custom_plugin
  ```

  <br>
  플러그인을 활성화시켜 주세요.

  ```
  php artisan plugin:activate custom_plugin
  ```

  </p>
</details>
